\# GitOps Cluster Configuration



Configuration GitOps pour mon cluster Kubernetes Talos avec ArgoCD.



\## Applications déployées



1\. \*\*app1-nginx\*\* : Serveur web Nginx

2\. \*\*app2-redis\*\* : Base de données Redis

3\. \*\*app3-guestbook\*\* : Application de démonstration



\## Architecture



\- 3 Control Plane nodes

\- 2 Worker nodes

\- ArgoCD pour GitOps

\- Cilium CNI

\- MetalLB LoadBalancer
